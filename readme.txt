== Description ==
The ultimate add-on for Contact Form 7 – redirect to any page you choose or any external link after the form submits successfully.
*This plugin requires Contact Form 7 version 4.8 or later.

Features of **NAAS Contact Form 7 Redirect** Plugin
1. Redirect to any URL
2. Open URL in a new tab
3. Redirect to any selected pages

== Installation ==
Installing NAAS Contact form 7 redirect can be done either by searching for “NAAS Contact Form 7 Redirect” via the “Plugins > Add New” screen in your WordPress dashboard, or by using the following steps:
1. Download the plugin via WordPress.org.
2. Upload the ZIP file through the “Plugins > Add New > Upload” screen in your WordPress dashboard.
3. Activate the plugin through the ‘Plugins’ menu in WordPress
4. Configure the settings in each form under NAAS Redirect Tab
